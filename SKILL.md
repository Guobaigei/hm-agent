# HM Agent

面向海绵系统的查询 Agent，适合品牌、公司、门店、项目等只读查询场景。

当前对外只暴露一个 tool：

- `hm-query(message)` - 输入自然语言查询，内部自动判断应该查询品牌、公司、门店还是项目，并以 Markdown 表格返回结果摘要

## 返回行为

- 如果命中唯一结果，会直接返回摘要
- 如果命中查询结果，`reply` 会使用 Markdown 表格，列为：实体类型、ID、名称、摘要、来源
- 如果命中多个候选对象，会返回 `needsClarification=true` 和候选项表格
- `results` 会保留结构化查询结果，方便调用方二次处理
- 如果海绵接口请求失败，会返回明确的失败原因，而不是伪装成“未查询到”

## 适合的自然语言示例

- 帮我查一下肯德基这个品牌
- 查一下上海的肯德基门店
- 看看肯德基相关的项目
- 帮我找一下肯德基公司的信息
